# AI-Powered Resume Screening & Job Matching System

## Features
- Resume Upload
- Resume Parsing
- Skill Extraction
- Resume Categorization using SVM
- Job Recommendation using BERT
- Flask Web Interface

## Technologies Used
- Python
- Flask
- Scikit-learn
- Sentence Transformers (BERT)
- HTML/CSS
- Pandas

## Project Structure

```bash
MINOR_SEM4/
│
├── app.py

├── utils/
├── notebook/
├── datasets/
├── requirements.txt
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```

## Output
- Predicts resume category
- Extracts skills
- Recommends jobs
